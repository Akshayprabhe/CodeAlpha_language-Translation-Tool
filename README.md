# 🌍 Language Translation Tool

A responsive web application built with Python and Streamlit that translates text between multiple languages in real-time using the Google Translator API.

## ✨ Features
- **Multi-Language Support**: Translate between 15+ languages including Hindi, Marathi, Tamil, Spanish, French
- **Instant Translation**: Fast and accurate results with 1 click
- **Clean UI**: Simple and user-friendly interface built with Streamlit
- **Easy to Deploy**: Runs locally with just 2 commands

## 🛠️ Tech Stack
- **Python** 3.14
- **Streamlit** - For web UI
- **deep-translator** - For translation API

## 🚀 How to Run Locally

1. **Clone the repository**
 git clone https://github.com/your-username/Translation-Tool.git
 cd Translation-Tool

2.**Install dependencies**
 pip install -r requirements.txt
 Run the app
 streamlit run app.py
